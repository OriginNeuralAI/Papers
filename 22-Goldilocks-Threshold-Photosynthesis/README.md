# The Goldilocks Threshold
**Quantitative Limits of Evolutionary Optimization in Quantum Energy Transport**

A critical complexity threshold N_c ≈ 5 governs which quantum energy funnels evolution can design. The Fenna-Matthews-Olson complex (N=7) sits precisely at the evolutionary accessibility boundary — the largest system where directed search succeeds in tractable time.

**Key findings:**
- Reliability sigmoid: R(N) = 1/(1 + exp(0.50 × (N − 5.0))), fit error 5.5×10⁻⁴
- FMO (N=7): 26.7% random reliability, directed evolution finds funnel in ~21 generations
- LH2 (N=27): 0.0% reliability under ALL tested strategies (symmetry, hierarchy, modularity)
- No spectral shortcut exists: all Pearson correlations |r| < 0.07
- Funnel reliability is irreducibly global — the landscape must be traversed to be evaluated
