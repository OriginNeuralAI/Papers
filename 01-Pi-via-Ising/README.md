# Computing Pi via Ising Model Ground States
**Three Novel Methods Using Combinatorial Optimization**

Three independent methods for computing pi using Ising model ground state optimization — the first such computations. Method 1 (Gauss Lattice): 7 correct decimal digits via Richardson extrapolation over 14 GPU-accelerated solves. Method 2 (BBP-QUBO): 32 correct hexadecimal digits. Method 3 (QUBO Modular Exponentiation): 8/8 correct hex digits from scratch — the ground state IS the computation.
