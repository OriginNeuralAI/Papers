# Moonshine Corrections to Arithmetic Functions
**An Empirical Test**

Tests two moonshine-derived formulas against known data:

1. **Twin Prime Correction (v2)**: Monster prime Gaussian weight w_M'(x; sigma=2.5) outperforms Hardy-Littlewood (MAE 11.0% vs 12.7%), with 7x error reduction at x=10^3. Does not beat Li_2 (MAE 3.2%).
2. **Busy Beaver Bounds (Variant C)**: c_n^{0.4} * |spectral| achieves mean |ln(ratio)| = 2.63, approximating BB(3) within factor 1.4.

Original formulas (v1) diagnosed and fixed: j-function scales were astronomical; Tr/|M| killed the signal. The corrected parameterizations produce genuine improvements. Honest negative results reported alongside positives.
