# The U24 Programme: Six Millennium Problems and the Non-Polynomial Obstruction

Framework connecting six Clay Millennium Prize Problems through the universality constant Omega = 24: Riemann Hypothesis, Yang-Mills mass gap, BSD conjecture, P vs NP, Navier-Stokes regularity, and Hodge conjecture. Each problem addressed through the spectral operator framework with honest epistemic marking.
