**Hitting Sets for CM Discriminants and Coupling Constants of Fermat Curves**

**Status: Contains unconditional proofs (finite computation)**

1. **Optimal CM Hitting Set** (Proved): S = {5, 11, 17, 47} separates all 9 class-number-one discriminants. No 3-prime set works (exhaustive over 15,180 triples). Extended to h=2 (optimal size 5) and h=3.
2. **Coupling Constants** (Computed): 15 values for Fermat curves x^n + y^n = 1 over F_p, through n=47 and primes to p=50,000. Scaling: alpha ~ 0.76 * n^0.33.

Additionally: Ising ground state on discriminant landscape (1,526 discriminants) naturally separates class numbers.
